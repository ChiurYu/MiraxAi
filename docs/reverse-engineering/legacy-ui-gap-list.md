# 旧版轻语 IP 界面差距清单

本文记录旧版「轻语 IP 智能体」可见界面与当前 Mirax AI 桌面第一版之间的差距。状态用于驱动后续任务拆分和验收。

## 已对齐

- 左侧导航：首页、声音管理、形象管理、素材管理、任务中心、账号管理、设置。
- 顶部模式：手动、自动、后台。
- 首页工作台：7 个编号模块，覆盖学习对标、改写文案、声音生成、视频生成、一键成片、标题封面、视频发布。
- 用户自行配置模型密钥，API Key 仅保存在运行态，不写入本地草稿。
- 一键跑通 mock 流程，能生成音频、数字人视频、成片、封面和 mock 发布任务。

## 第一版必须补齐

| 旧版区域 | 当前问题 | 完成标准 | 状态 |
| --- | --- | --- | --- |
| 学习对标 | 粘贴和上传仍是路径输入 / prompt fallback，缺少明确的素材选择状态 | Tauri 可用时打开系统文件选择；Web fallback 保留路径输入；卡片展示已选来源 | 未开始 |
| 改写文案 | 改写结果没有独立状态标识，失败后用户不容易判断是否可重试 | 卡片显示 pending/running/completed/failed 状态 badge，并提供对应步骤重试按钮 | 未开始 |
| 声音生成 | 上传声音只是路径输入，克隆声音和合成语音的 mock 边界不够清楚 | 使用统一 PathPickerButton；展示音频路径、声音 ID 和 mock 标识 | 未开始 |
| 视频生成 | 单卡片生成按钮与全流程按钮边界不清，失败恢复路径不明显 | 单卡片按钮只执行对应步骤；失败时保留错误信息和重试入口 | 未开始 |
| 一键成片 | 字幕、BGM、音量控件只展示，不参与 mock 输出摘要 | mock render 日志包含字幕、BGM、音量、剪气口等设置摘要 | 未开始 |
| 标题封面 | 话题标签没有解析为数组，封面导出只是路径展示 | 发布前把逗号分隔标签转为 `#tag` 预览；封面区展示生成 / 导出状态 | 未开始 |
| 视频发布 | 没有发布前确认，也没有任务中心历史记录 | 发布前展示账号、平台、模式、视频路径；发布后写入任务中心记录 | 未开始 |
| 设置 | 密钥只做字段校验，不显示连接测试结果 | Mock 连接测试成功；OpenAI-compatible 明确显示“真实接入未启用” | 未开始 |
| 依赖 | sidecar 检查只在包内，UI 没有入口 | UI 展示 FFmpeg、Playwright、Python、HeyGem、CosyVoice 状态和处理建议 | 未开始 |

## 非第一版范围

- 真实 Whisper / CosyVoice / HeyGem 服务编排。
- 真实平台自动发布登录态托管。
- 多项目 SQLite 数据迁移和完整任务队列。
- 旧版所有素材库、账号库、形象库的完整管理页面。
