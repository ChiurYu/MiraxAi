# 功能演示视频时间轴

源文件：`功能演示视频.mp4`

视频时长约 128 秒。以下时间轴基于关键帧抽取和画面观察整理，用于第一版桌面端重建的可见功能复刻清单。后续实现时，如果需要更精确的文案和点击顺序，应结合原视频逐帧复核。

## 总体判断

演示视频展示的是一条完整的口播视频自动化生产链路：

1. 展示最终生成的视频效果。
2. 进入软件主页，选择对标视频或模板。
3. 自动提取对标文案。
4. 一键 AI 改写文案。
5. 使用自己的数字人形象出镜。
6. 自动生成字幕。
7. 生成标题、封面、话题、描述等发布信息。
8. 选择发布账号和平台。
9. 自动跳转平台发布页并上传视频。

## 页面与功能清单

第一版需要复刻以下页面或工作区。

### 1. 对标内容导入

- 支持输入视频链接。
- 支持从素材墙或历史对标列表选择视频。
- 支持展示对标视频缩略图、标题、来源平台。
- 支持提取视频文案。

### 2. 文案提取与改写

- 显示原始提取文案。
- 支持“一键改写文案”。
- 支持 AI 自动仿写和按指令仿写。
- 支持改写结果回填。
- 支持手动编辑文案。

### 3. 声音与数字人配置

- 支持选择声音 / 音色。
- 支持选择自己的数字人形象。
- 支持展示数字人预览图或视频。
- 支持进入下一步生成。

### 4. 视频生成与字幕

- 支持生成数字人口播视频。
- 支持自动生成字幕。
- 支持视频预览。
- 支持显示生成进度和状态。

### 5. 标题、封面与发布信息

- 支持生成标题。
- 支持生成或选择封面。
- 支持封面模板设置。
- 支持描述、标签、话题等发布信息。

### 6. 账号管理与发布

- 支持选择发布账号。
- 支持显示平台和账号登录状态。
- 支持直接发布或保存草稿。
- 支持发布后关闭浏览器等选项。
- 支持打开平台后台并自动上传视频。

## 对实现计划的影响

第一版实现应优先保证主流程闭环，而不是一次性实现所有隐藏功能。

建议任务顺序：

1. 搭建桌面端工作台骨架。
2. 实现对标链接输入和文案提取的模拟接口。
3. 实现 AI Provider 配置和文案改写。
4. 实现声音、数字人、模板的选择 UI。
5. 实现 FFmpeg 视频合成的最小可用链路。
6. 实现封面和发布信息配置。
7. 实现账号管理和发布任务模型。
8. 接入 Playwright 发布自动化。

## 待复核点

- 原视频中具体平台图标和账号列表文字需要后续高清逐帧确认。
- “一键追爆款并发布到各平台”是否作为单独快捷按钮，需要结合旧 Python 原型和完整 UI 进一步确认。
- 快手是否进入第一版主流程，取决于视频中是否有明显展示；目前建议优先抖音、小红书、视频号。
