## 可见控件

| 控件 | 类型 | 启用状态 | 观察到的行为 |
| --- | --- | --- | --- |
| 声音训练 卡片 | card | 启用 | 提供上传音频/视频或直接录音两种入口，训练专属声音模型 |
| 上传音频 标签 | tab | 选中（默认） | 展示文件拖拽上传区 |
| 直接录音 标签 | tab | 未选中 | 截图未展示录音界面 |
| 文件拖拽上传区 | dropzone | 启用 | 提示「点击或拖拽音频/视频文件到此区域」，支持 MP3/WAV/M4A/MP4/MOV 等格式，视频会自动提取音频；建议时长 5-30 秒，文件大小不超过 50MB |
| 声音名称 输入框 | text | 启用 | placeholder「给这个声音起个名字」 |
| 开始训练 按钮 | button | 启用 | 点击后应开始声音克隆训练；截图未展示训练过程或结果 |
| 声音合成 卡片 | card | 启用 | 选择已训练的声音模型并输入文案生成语音 |
| 选择声音模型 下拉框 | select | 启用 | 默认无选中项或显示「请选择声音模型」类占位文案 |
| 输入文案 文本域 | textarea | 启用 | 截图中示例文案为「你好，这是一个声音克隆测试。」，右下角显示字符数 18 / 5000 |
| 语速 滑块 | slider | 启用 | 用于调整合成语音语速，当前位置在截图中可见 |
| 情绪类型 / 情绪强度 下拉框 | select | 启用 | 截图中显示选中值「0.5」，具体含义待确认 |
| 生成语音 按钮 | button | 启用 | 点击后应生成语音；截图未展示试听或下载入口 |
| 我的声音库 标题区 | section header | 启用 | 显示副标题「查看你训练过的所有声音模型」，右侧有刷新按钮 |
| 声音卡片 | card | 启用 | 每个卡片展示声音封面/播放按钮、声音名称、描述、标签、创建时间和编辑/删除图标 |

## 表单字段

| 字段 | 输入类型 | 默认值或示例值 | 校验或帮助文案 |
| --- | --- | --- | --- |
| 训练音频/视频 | 文件拖拽/选择 | 空 | 支持 MP3, WAV, M4A, MP4, MOV 等格式；建议时长 5-30 秒；文件大小不超过 50MB；视频会自动提取音频 |
| 声音名称 | text | placeholder「给这个声音起个名字」 | 必填；每个账号默认仅支持 5 个声音，如需更多请联系客服 |
| 选择声音模型 | select | 未选择 | 从「我的声音库」中已训练的声音模型中选择 |
| 输入文案 | textarea | 示例「你好，这是一个声音克隆测试。」 | 最多 5000 字 |
| 语速 | slider | 中间位置 | 调节合成语速 |
| 情绪类型 / 情绪强度 | select | 截图显示「0.5」 | 具体枚举值和含义待确认 |
