# PAGE-HOME-WORKBENCH：首页工作台

## 身份信息

| 字段 | 值 |
| --- | --- |
| 页面 ID | PAGE-HOME-WORKBENCH |
| 页面名称 | 首页工作台 |
| 旧版入口路径 | 启动旧版 App 后默认首页，或左侧导航「首页」 |
| 巡检优先级 | P0 |

## 页面区域

| 区域 | 可见内容 | 备注 |
| --- | --- | --- |
| 左侧导航 | 首页、形象管理、素材管理、创作管理、任务中心、账号管理、设置、帮助 | 不评价旧版视觉风格 |
| 顶部区域 | 手动 / 自动 / 草稿 三个模式切换标签；右侧为账号/设置相关入口 | 只记录信息结构 |
| 主工作区 | 7 个编号模块：1.学习对标、2.改写文章、3.声音生成、4.视频生成、5.一键成片、6.标题封面、7.视频发布 | 对应核心生产流程；右侧有视频预览区 |

## 可见控件

| 控件 | 类型 | 启用状态 | 观察到的行为 |
| --- | --- | --- | --- |
| 手动 / 自动 / 草稿 标签 | tab | 手动 默认选中 | 切换顶部模式，影响主工作区可用流程 |
| 视频链接输入框 | input | 启用 | 用于粘贴对标视频链接 |
| 提取文案按钮 | button | 启用 | 点击后应触发视频解析/文案提取 |
| 改写文章按钮 | button | 启用 | 点击后应基于提示词改写文章 |
| 生成声音按钮 | button | 启用 | 点击后应基于选定声音合成语音 |
| 生成视频按钮 | button | 启用 | 点击后应基于选定形象生成数字人视频 |
| 一键成片按钮 | button | 启用 | 点击后应基于配置合成完整成片 |
| 视频发布按钮 | button | 启用 | 点击后应进入发布前确认或发布流程 |
| 选择声音 / 上传声音 / AI 克隆声音 | button / select | 启用 | 声音样本来源选择 |
| 选择形象 / 上传形象 | button / select | 启用 | 数字人形象来源选择 |

## 表单字段

| 字段 | 输入类型 | 默认值或示例值 | 校验或帮助文案 |
| --- | --- | --- | --- |
| 视频链接 | text | 空 | 提示「请输入视频链接」 |
| 改写提示词 | textarea | 空 | 可输入改写风格或要求 |
| 字数 | number / select | 默认 200 字 | 控制改写文案长度 |
| 选择声音 | select / dropdown | 默认声音列表 | 显示默认声音样本名称 |
| 选择形象 | select / dropdown | 默认形象列表 | 显示默认数字人形象名称 |
| 字幕开关 | toggle | 开启 | 需先设置关键字才能使用模板 |
| BGM 音量 | slider | 0.3 | 控制背景音乐音量 |
| 人声音量 | slider | 1 | 控制合成语音音量 |

## 弹窗与提示

| 触发方式 | 弹窗或提示 | 操作按钮 | 结果 |
| --- | --- | --- | --- |
| 启动 App 时 token 失效 | 登录页提示「登录已过期或权限不足，请重新登录」 | 登录 / 注册 / 忘记密码 | 需要账号登录才能进入首页 |
| 启动 App 时未激活会员 | 弹窗提示「请输入激活码激活会员以使用完整功能」 | 取消 / 激活 | 需要激活码才能使用完整功能 |

## 状态变化

| 操作 | 操作前 | 操作后 | 证据 ID |
| --- | --- | --- | --- |
| 启动 App | 应用启动 | 尝试自动登录，失败后跳转登录页并显示激活弹窗 | EV-RUNTIME-001 |
| 登录成功（推断） | 登录页 | 进入首页工作台，显示完整生产流程 | EV-RUNTIME-001 |
