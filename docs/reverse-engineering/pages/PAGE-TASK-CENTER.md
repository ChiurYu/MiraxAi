## 状态变化

| 操作 | 操作前 | 操作后 | 证据 ID |
| --- | --- | --- | --- |
| 点击新建任务 | 空任务列表 | 进入新建任务流程 | EV-RUNTIME-130（推断） |
| 选择筛选条件并点击筛选 | 全部任务 | 显示符合条件的任务 | EV-RUNTIME-130（推断） |
| 点击清空 | 已应用筛选 | 重置为默认全部任务 | EV-RUNTIME-130（推断） |
| 任务执行中 | 进行中 0 | 进度条、当前步骤更新 | EV-RUNTIME-130（推断） |
| 任务完成/失败 | 进行中 | 状态变为已完成/失败，操作列出现查看/重试入口 | EV-RUNTIME-130（推断） |

## 可执行动作

| 动作 | 结果 | 关联功能卡 |
| --- | --- | --- |
| 新建任务 | 创建一条短视频生产任务 | FC-HOME-PIPELINE、FC-ASSET-MANAGEMENT |
| 刷新任务列表 | 获取最新任务状态 | FC-ASSET-MANAGEMENT |
| 筛选任务 | 按状态或执行模式定位任务 | FC-ASSET-MANAGEMENT |
| 查看任务详情/结果 | 打开任务产物或详细日志 | FC-ASSET-MANAGEMENT |
| 重试失败任务 | 重新执行失败步骤 | FC-ASSET-MANAGEMENT |
| 删除任务 | 从任务中心移除任务记录 | FC-ASSET-MANAGEMENT |

## 受限动作

| 动作 | 障碍 | 运行障碍 ID |
| --- | --- | --- |
| 创建并执行真实生产任务 | 需要 AI 模型、声音克隆、数字人、渲染、发布等完整链路，当前旧版 App 受登录/激活/云服务/本地依赖限制 | RB-ASSET-004 |
| 查看任务进度、日志、失败原因和产物 | 当前任务列表为空，真实状态流转未验证 | RB-ASSET-004 |
| 重试失败任务 | 需要前置步骤可独立重跑；截图未展示重试入口 | RB-ASSET-004 |
