## 页面区域

| 区域 | 可见内容 | 备注 |
| --- | --- | --- |
| 发布入口 | 首页工作台第 7 步「视频发布」卡片 | 与第 6 步「标题封面」相邻，发布字段集中在首页主工作区 |
| 发布前准备区 | 第 6 步「标题封面（用于发布）」：标题、描述、话题标签、视频封面 | 标题/描述/话题/封面在发布前完成准备 |
| 账号管理入口 | 左侧导航「账号管理」；首页第 7 步「发布账号」下拉框 | 账号页提示「已登录成功的账号才会在发布时可以选择」 |
| 任务中心入口 | 左侧导航「任务中心」；可查看发布任务状态、进度、失败原因 | 当前截图中任务列表为空，未观察到已产生的发布任务 |

## 可见控件

| 控件 | 类型 | 启用状态 | 观察到的行为 |
| --- | --- | --- | --- |
| 视频地址 输入框 | input | 启用 | 用于填写或选择待发布视频地址 |
| 发布账号 下拉框 | select / dropdown | 启用 | 显示可选平台账号；当前无账号时可能为空或提示添加账号 |
| 发布方式 单选 / 切换 | radio / toggle | 启用 | 「直接发布」与「草稿」两种模式 |
| 立即发布 按钮 | button | 启用 | 点击后应创建发布任务或进入确认/授权流程 |
| 添加账号 按钮 | button | 启用 | 位于账号管理页，点击后应进入平台账号授权或登录流程 |
| 任务中心 筛选器 | select / button | 启用 | 可按任务状态、执行模式筛选，查看发布任务结果 |
